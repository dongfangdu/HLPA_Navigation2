启动说明
----------

* 构建docker镜像
```
# 删除文件夹
find . -name 'dirs.tar' -exec rm {} -f \;


# 执行源码打包脚本
bash run_docker_before.sh


# 重新构建镜像
docker-compose -f docker-compose.dev.yml build


```

* 运行全部容器
```
# 运行
docker-compose -f docker-compose.dev.yml up -d

```
